macOS — Xcode Command Line Tools

La méthode la plus simple sur macOS. Installe clang++ (compatible C++ moderne).

Étapes

  • Ouvrez Terminal (Spotlight → « Terminal »).
  • Lancez: xcode-select --install
  • Acceptez la licence et attendez la fin.

Vérifier

clang++ --version
# ou
c++ --version
Remarque : sur macOS, la commande g++ pointe souvent vers clang++.

macOS — Homebrew (optionnel)

Utile si vous voulez une version spécifique de GCC ou de LLVM.

Étapes

  • Installez Homebrew: brew.sh
  • Installez LLVM (clang++ récent): brew install llvm
  • Optionnel: ajouter au PATH (Apple Silicon): echo 'export PATH="/opt/homebrew/opt/llvm/bin:$PATH"' >> ~/.zshrc
Alternative : brew install gcc fournit g++-14 (numéro selon la version).

Windows — Visual Studio (gratuit)

Installe le compilateur MSVC et tous les outils nécessaires.

Étapes

  • Téléchargez Visual Studio Community (ou Build Tools).
  • Choisissez la charge de travail: Développement Desktop en C++.
  • Ouvrez Developer Command Prompt for VS.

Vérifier

cl
Astuce : utilisez ce terminal spécial pour que cl soit disponible dans le PATH.

Windows — MSYS2 + MinGW

Alternative légère pour g++ (GNU). Recommandé si vous préférez GCC.

Étapes

  • Téléchargez MSYS2 : msys2.org
  • Ouvrez MSYS2 MSYS et mettez à jour : pacman -Syu (redémarrez si demandé)
  • Installez l’outilchain 64-bit : pacman -S --needed base-devel mingw-w64-ucrt-x86_64-toolchain
  • Ouvrez le terminal MSYS2 MinGW UCRT64.

Vérifier

g++ --version

Windows — Code::Blocks (avec MinGW)

Alternative « tout-en-un » qui installe Code::Blocks + MinGW.

Étapes

  • Téléchargez l’installateur avec MinGW (ex : codeblocks-20.03mingw-setup.exe).
  • Installez en laissant les options par défaut.
  • Créez un projet « Console application » → C++ → Compilez et exécutez.

Vérifier

g++ --version
where g++
Conseil : si vous avez déjà MSYS2/MinGW, vous pouvez configurer Code::Blocks pour utiliser ce compilateur.

VS Code — Éditeur conseillé

Installation

  • Installez Visual Studio Code.
  • Ajoutez l’extension C/C++ (Microsoft).

Compilation simple (Terminal intégré)

  • Ouvrez un dossier → créez main.cpp.
  • Terminal → exécutez : 
    // macOS (clang++)
clang++ main.cpp -std=c++17 -O2 -o main && ./main

// Windows (MSVC)
cl /EHsc /std:c++17 main.cpp && main.exe

// Windows (MinGW)
g++ main.cpp -std=c++17 -O2 -o main.exe && ./main.exe

Test rapide — Hello World

#include <iostream>

int main() {
    std::cout << "Bonjour C++!" << std::endl;
    return 0;
}

Compilez et lancez selon votre plateforme (voir diapo précédente).

Si cela affiche « Bonjour C++! » → tout est prêt pour le cours.

Dépannage

  • commande introuvable : ouvrez le bon terminal (Developer Prompt VS, MSYS2 UCRT64, Terminal macOS).
  • cl introuvable : lancez Developer Command Prompt for VS.
  • g++ introuvable (Windows) : installez MSYS2 + toolchain, puis utilisez le terminal MinGW UCRT64.
  • Permission refusée (macOS) : relancez l’installation xcode-select --install.
  • VS Code ne trouve pas le compilateur : compilez via le terminal intégré plutôt que le bouton « Run ».
