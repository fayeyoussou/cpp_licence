Examen : Atelier de Couture

Description des classes et fonctionnalites - POO C++

Vue d'ensemble

  • Personne : classe abstraite de base
  • Client : derive de Personne
  • Couturier : derive de Personne (classe de base pour les types de couturiers)
  • CouturierSenior : derive de Couturier
  • Apprenti : derive de Couturier
  • Tissu : materiau pour les habits
  • Commande : lie un client a un tissu
  • HabilleTermine : habit realise (classe final)
  • Atelier : classe principale qui gere tout

Hierarchie d'heritage

Personne (abstraite)
    |
    +-- Client
    |
    +-- Couturier
            |
            +-- CouturierSenior
            |
            +-- Apprenti

Relations

Heritage (est-un)

  • Client est-une Personne
  • Couturier est-une Personne
  • CouturierSenior est-un Couturier
  • Apprenti est-un Couturier

Composition (a-un)

  • Atelier a-une liste de Client
  • Atelier a-une liste de Couturier (peut contenir CouturierSenior et Apprenti)
  • Atelier a-une liste de Tissu
  • Atelier a-une liste de Commande
  • Atelier a-une liste de HabilleTermine
  • Commande a-un Client
  • Commande a-un Tissu
  • HabilleTermine a-une Commande
  • HabilleTermine a-un Couturier

Classe Personne (abstraite)

Attributs (protected)

  • id : identifiant unique (entier)
  • nom : nom de la personne (chaine)
  • telephone : numero de telephone (chaine)

Methodes

  • afficher() : methode virtuelle pure, affiche les infos sur un flux
  • id(), nom(), telephone() : accesseurs
  • setNom(), setTelephone() : mutateurs

Operateurs

  • << : affiche la personne
  • >> : saisit une personne
  • ==, != : compare par id

Particularites

  • Classe abstraite (methode virtuelle pure)
  • Destructeur virtual
  • Attributs protected

Classe Client

Heritage

  • Client est-une Personne
  • herite : id, nom, telephone

Attributs propres (private)

  • adresse : adresse du client (chaine)
  • nombreCommandes : compteur de commandes (entier)

Methodes

  • afficher() : redefinition avec override
  • adresse(), nombreCommandes() : accesseurs
  • setAdresse() : mutateur
  • incrementerCommandes() : augmente le compteur de 1

Operateurs

  • << : affiche le client
  • >> : saisit un client

Classe Couturier

Heritage

  • Couturier est-une Personne
  • herite : id, nom, telephone

Attributs propres (protected)

  • specialite : domaine d'expertise (chaine)
  • tarifHoraire : prix par heure (reel)

Methodes

  • afficher() : redefinition avec override
  • calculerSalaire(heures) : methode virtuelle, retourne heures * tarifHoraire
  • specialite(), tarifHoraire() : accesseurs
  • setSpecialite(), setTarifHoraire() : mutateurs

Operateurs

  • << : affiche le couturier
  • >> : saisit un couturier

Particularites

  • Attributs protected pour les classes derivees
  • calculerSalaire() est virtuelle pour permettre la redefinition

Classe CouturierSenior

Heritage

  • CouturierSenior est-un Couturier
  • herite : id, nom, telephone, specialite, tarifHoraire

Attributs propres (private)

  • bonus : bonus mensuel (reel)
  • anneesExperience : annees d'experience (entier)

Methodes

  • afficher() : redefinition avec override
  • calculerSalaire(heures) : redefinition avec override, ajoute le bonus au salaire de base
  • bonus(), anneesExperience() : accesseurs
  • setBonus() : mutateur

Operateurs

  • << : affiche le couturier senior
  • >> : saisit un couturier senior

Classe Apprenti

Heritage

  • Apprenti est-un Couturier
  • herite : id, nom, telephone, specialite, tarifHoraire

Attributs propres (private)

  • mentor : pointeur vers le couturier mentor
  • dateDebut : date de debut de l'apprentissage (chaine)

Methodes

  • afficher() : redefinition avec override
  • calculerSalaire(heures) : redefinition avec override, applique un coefficient reduit (50%)
  • mentor(), dateDebut() : accesseurs
  • setMentor() : mutateur

Operateurs

  • << : affiche l'apprenti
  • >> : saisit un apprenti

Classe Tissu

Attributs (private)

  • id : identifiant (entier)
  • nom : type de tissu (chaine)
  • couleur : couleur du tissu (chaine)
  • prixAuMetre : prix par metre (reel)
  • quantiteStock : metres en stock (reel)

Methodes

  • id(), nom(), couleur(), prixAuMetre(), quantiteStock() : accesseurs
  • setPrixAuMetre() : mutateur
  • ajouterStock(quantite) : ajoute au stock
  • retirerStock(quantite) : retire du stock

Operateurs

  • << : affiche le tissu
  • >> : saisit un tissu
  • ==, != : compare par id

Classe Commande

Attributs (private)

  • id : identifiant (entier)
  • client : pointeur vers le client
  • tissu : pointeur vers le tissu
  • metresNecessaires : quantite de tissu (reel)
  • description : description de l'habit (chaine)
  • terminee : etat de la commande (booleen)

Methodes

  • id(), client(), tissu(), metresNecessaires(), description(), terminee() : accesseurs
  • setTerminee() : marque la commande comme terminee
  • calculerPrix() : retourne metresNecessaires * prixAuMetre

Operateurs

  • << : affiche la commande
  • >> : saisit une commande
  • == : compare par id

Classe HabilleTermine (final)

Attributs (private)

  • id : identifiant (entier)
  • commande : pointeur vers la commande
  • couturier : pointeur vers le couturier
  • dateTerminaison : date de fin (chaine)
  • heuresTravail : heures de travail (entier)

Methodes

  • id(), commande(), couturier(), dateTerminaison(), heuresTravail() : accesseurs
  • calculerCoutTotal() : retourne prixCommande + salaireCouturier

Operateurs

  • << : affiche l'habit termine

Particularites

  • Classe final : ne peut pas etre derivee

Classe Atelier

Attributs (private)

  • nom : nom de l'atelier (chaine)
  • clients : liste de clients (vector)
  • couturiers : liste de pointeurs vers Couturier (pour eviter le slicing)
  • tissus : liste de tissus (vector)
  • commandes : liste de commandes (vector)
  • habitsTermines : liste d'habits termines (vector)

Methodes

  • ajouterClient(), trouverClient(), supprimerClient() : gestion des clients
  • ajouterCouturier(), trouverCouturier() : gestion des couturiers
  • ajouterTissu(), trouverTissu() : gestion des tissus
  • ajouterCommande(), trouverCommande() : gestion des commandes
  • ajouterHabilleTermine() : ajoute un habit termine
  • afficherTousCouturiers() : parcourt la liste et appelle afficher() sur chaque couturier

Operateurs

  • << : affiche l'atelier
  • [] : retourne un client par son id

Checklist

  • Classe abstraite avec = 0
  • Heritage simple (Personne -> Client, Couturier)
  • Sous-heritage (Couturier -> CouturierSenior, Apprenti)
  • Classe final (HabilleTermine)
  • override sur les redefinitions
  • Destructeur virtual dans les classes de base
  • Vector de pointeurs pour eviter le slicing
  • Polymorphisme avec calculerSalaire()
  • Operateurs <<, >>, ==, !=, []
  • Listes d'initialisation
  • Attributs protected pour l'heritage
