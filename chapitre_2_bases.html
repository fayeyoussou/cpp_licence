Variables, types, opérateurs, contrôle, tableaux, chaînes

Types de base

  • int: entiers (ex: -3, 0, 42)
  • float, double: nombres à virgule (double = plus précis)
  • char: caractère (ex: 'A'), stocke un code (ASCII/UTF-8)
  • bool: logique, true ou false
  • Signé/Non signé: unsigned int (≥ 0)
  • Tailles typiques: int 32 bits, double 64 bits (selon plateforme)
#include <iostream>
int main() {
    int entier = 42;
    double reel = 3.14;    // préférez double à float au début
    char lettre = 'A';
    bool ok = true;
    std::cout << entier << " " << reel << " " << lettre << " " << ok << "\n";
}
Important: les guillemets simples 'A' pour un char, doubles "ABC" pour une chaîne de caractères.

Littéraux et notations

  • Entiers: décimal 42, hex 0x2A, binaire 0b101010
  • Réels: 3.14, suffixe f pour float (ex: 1.5f)
  • Caractères: 'A', échappements '\n' (nouvelle ligne), '\\'
  • Booléens: true, false
int x = 42; unsigned u = 42u; double d = 1.0; float f = 1.0f;
char nl = '\n'; char backslash = '\\';
Astuce: utilisez les suffixes (u, f) quand nécessaire pour préciser le type.

Variables et constantes

  • Déclaration: type nom; (ex: int age;)
  • Initialisation: = (copie), () (directe), {} (uniforme)
  • const: valeur qui ne change pas (ex: const double PI = 3.14159;)
  • Noms clairs: surfaceTotale, noteFinale (évitez a1, x)
#include <iostream>
int main() {
    int age = 20;          // déclaration + initialisation
    const double TAUX = 1.2; // constante
    int a = 1; int b(2); int c{3};
    std::cout << a+b+c << " " << TAUX;
}
Astuce débutant: initialisez vos variables dès la déclaration pour éviter les valeurs indéfinies.

Portée des variables (scope)

  • Portée d'un bloc: entre { et }
  • Une variable locale n'existe qu'à l'intérieur de son bloc
  • Évitez d'utiliser la même variable dans des blocs imbriqués
#include <iostream>
int main(){
    int x = 1;
    {
        int x = 2; // masque le x extérieur
        std::cout << x; // 2
    }
    std::cout << x; // 1
}
Conseil: déclarez les variables au plus près de leur utilisation.

Opérateurs arithmétiques

  • +, -, *, /, % (modulo sur entiers)
  • Division entière tronque (ex: 7/2 == 3)
  • Incrément/décrément: ++, --; composés: +=, *=, ...
int main() {
    int a = 7, b = 2;
    std::cout << a / b << "\n";      // 3 (entier)
    std::cout << a / 2.0 << "\n";  // 3.5 (double)
    a += 5; // 12
}

Priorité des opérateurs

  • *, /, % avant +, -
  • Utilisez des parenthèses pour la clarté
  • Attention: = (affectation) ≠ == (égalité)
int r1 = 2 + 3 * 4;     // 14
int r2 = (2 + 3) * 4;   // 20
if (x == 3) { /* ok */ }
// if (x = 3) { ... }  // erreur logique fréquente
Astuce: ajoutez des parenthèses même si la priorité suffit: votre futur vous vous dira merci.

Relationnels et logiques

  • Relationnels: ==, !=, <, <=, >, >=
  • Logiques: && (ET), || (OU), ! (NON), avec court-circuit
bool adulte = (age >= 18);
if (x != 0 && y/x > 2) { /* x n'est pas 0 avant la division */ }
Rappel: une expression logique vaut true (1) ou false (0).

Conversions implicites et explicites

  • Promotion implicite (ex: int → double) lors de calculs
  • Rétrécissement = perte d’info (ex: 3.7 → 3)
  • Cast explicite recommandé: static_cast<type>(expr)
double d = 3;              // OK: 3.0
 int i{3.7};                // Erreur de compilation (narrowing)
 int j = static_cast<int>(3.7); // OK: j == 3
À éviter: les casts style C ((int)x) — préférez static_cast plus sûr et lisible.

Conversions — erreurs fréquentes

  • Mélange unsigned et int peut surprendre (sous-flux)
  • Perte de précision de double → int
  • Division entière inattendue si les deux opérandes sont entiers
unsigned ua = 1; int b = -2;
// b est converti en unsigned avant l'addition
auto s = ua + b; // valeur très grande selon la plateforme

int q1 = 5/2;     // 2
double q2 = 5/2;  // 2 (toujours entier avant conversion!)
double q3 = 5/2.0; // 2.5
Règle: forcez le type voulu (ex: 2.0) pour éviter la division entière.

Exercices — Bases

Calculatrice simple

#include <iostream>
int main() {
    double a{}, b{}; char op{};
    std::cout << "Entrez une opération (ex: 3 + 4): ";
    std::cin >> a >> op >> b;
    if (op == '+') std::cout << (a + b);
    else if (op == '-') std::cout << (a - b);
    else if (op == '*') std::cout << (a * b);
    else if (op == '/') {
        if (b != 0) std::cout << (a / b);
        else std::cout << "Erreur: division par 0";
    } else {
        std::cout << "Opérateur inconnu";
    }
}

Échange de variables

int a = 5, b = 9;
// Méthode avec variable temporaire (recommandée pour débuter)
int tmp = a; a = b; b = tmp; // a=9, b=5
// Ou bien avec la bibliothèque standard
// #include <utility>
// std::swap(a, b);

Structures conditionnelles

// if / else
if (note >= 10) {
    std::cout << "Admis";
} else if (note >= 8) {
    std::cout << "Rattrapage";
} else {
    std::cout << "Ajourné";
}

// switch
switch (jour) {
    case 1: std::cout << "Lundi"; break;
    case 2: std::cout << "Mardi"; break;
    default: std::cout << "Autre"; break;
}
Astuce: n'oubliez pas break dans chaque case pour éviter l'enchaînement non désiré.

switch en détail

  • Fonctionne avec int, char et énumérations
  • Pas de std::string (utilisez if/else)
  • Groupement de cases possible
char op; double a, b; std::cin >> a >> op >> b;
switch (op) {
    case '+': std::cout << (a+b); break;
    case '-': std::cout << (a-b); break;
    case '*': std::cout << (a*b); break;
    case '/': if (b!=0) std::cout << (a/b); else std::cout << "Div/0"; break;
    default: std::cout << "Opérateur inconnu"; break;
}

Boucles: for, while, do...while

for (int i = 0; i < 5; ++i) { std::cout << i; }
 int j = 5; while (j > 0) { --j; }
 int k = 0; do { ++k; } while (k < 3);
Quand utiliser quoi? for = nombre d'itérations connu; while = jusqu'à une condition; do...while = au moins une fois.

break, continue, return

for (int n = 1; n <= 10; ++n) {
     if (n % 2 == 0) continue; // saute pairs
     if (n > 7) break;          // arrête la boucle
     std::cout << n << " ";
 }
 return; // quitte la fonction
Conseil: utilisez return pour quitter tôt une fonction si une condition n'est pas remplie.

Exercices — Contrôle

  • Tables de multiplication (1..10)
  • Test de primalité (essais jusqu'à √n)
// Tables 1 à 10
 for (int i = 1; i <= 10; ++i) {
     for (int j = 1; j <= 10; ++j) {
         std::cout << (i * j) << "\t";
     }
     std::cout << "\n";
 }
 
 // Primalité
 bool est_premier(int n){
     if (n < 2) return false;
     for (int d = 2; d*d <= n; ++d)
         if (n % d == 0) return false;
     return true;
 }

Tableaux — Déclaration et parcours

int t[5] = {1,2,3,4,5}; // taille fixe connue
 for (int i = 0; i < 5; ++i) {
     std::cout << t[i] << " ";
 }
 // Calculer la taille (méthode simple)
 int taille = sizeof(t) / sizeof(t[0]);
Attention: pas de vérification automatique des indices; restez entre 0 et taille-1.

Tableaux — Saisie et bornes

#include <iostream>
int main(){
    const int MAX = 100; int t[MAX];
    int n; std::cout << "Combien d'éléments (max 100) ? "; std::cin >> n;
    if (n > MAX) { std::cout << "Tronqué à 100\n"; n = MAX; }
    for (int i = 0; i < n; ++i) std::cin >> t[i];
    // Afficher
    for (int i = 0; i < n; ++i) std::cout << t[i] << ' ';
}
Règle d'or: ne dépassez jamais la taille du tableau — comportement indéfini.

Tableaux multidimensionnels

int m[2][3] {{1,2,3},{4,5,6}};
 for (int i=0;i<2;++i)
   for (int j=0;j<3;++j)
     std::cout << m[i][j] << ' ';

Chaînes: char[] vs std::string

#include <iostream>
#include <string>
char cstr[] = "Bonjour";          // tableau de char terminé par '\0'
std::string s = "Hello";          // chaîne C++
s += ", world";                   // concaténation
std::cout << s.size();             // longueur

// Lecture d'une ligne entière (avec espaces)
std::string nom;
std::getline(std::cin, nom);
Conseil: préférez std::string au quotidien; utilisez char[] pour compatibilité C.

Parcours d'un tableau (exemples)

// Somme des éléments
int t[5] = {1,2,3,4,5};
int somme = 0;
for (int i = 0; i < 5; ++i) somme += t[i];
std::cout << "Somme = " << somme;

// Recherche d'un élément
int cible = 3; bool trouve = false;
for (int i = 0; i < 5; ++i) if (t[i] == cible) { trouve = true; break; }

Exercices — Tableaux et chaînes

  • Lire N nombres, les afficher et calculer la moyenne
  • Recherche d'un élément (retourner l'index ou -1)
  • Inverser une chaîne manuellement
// Inversion manuelle d'une chaîne std::string s
for (size_t i = 0, j = s.size() - 1; i < j; ++i, --j) {
    char tmp = s[i]; s[i] = s[j]; s[j] = tmp;
}

Récap & bonnes pratiques

  • Déclarez/initialisez clairement vos variables; utilisez const quand possible
  • Comprenez la différence int vs double et la division entière
  • Faites attention aux indices des tableaux (0 à n−1)
  • Préférez std::string pour les chaînes
  • Pratiquez avec les exercices pour ancrer les notions
